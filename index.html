Occlusion-Aware window segmentation with conditional adversarial training guided by structural prior information
Manuela F. Cerón Viveros
source code

This work integrates image segmentation with adversarial generative training to handle occlusions in façade images. OA-WinSeg uses rectangular boundary guidance to condition the generation of window segmentation masks, capturing structural information about hidden window boundaries. By incorporating adversarial training, rectangular guidance, and bidirectional convolution with 1D kernels, we achieve highly rectangular window shapes, reducing the reliance on complex vectorization mechanisms.
Simulated dataset with synthetic occlusions for façade images
Manuela F. Cerón Viveros
source code

This work presents a new method to generate synthetic datasets with occlusions. It enables configuring the percentage of occluded samples, setting size and location constraints for occluding objects. The approach is adaptable across building styles and image resolutions, supports adding new occluding object categories, and facilitates generating datasets of any scale. This dataset addresses challenges in manual occlusion annotation and enables automated real occlusion detection.
