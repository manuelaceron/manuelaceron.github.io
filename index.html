Occlusion-Aware window segmentation with conditional adversarial training guided by structural prior information
Manuela F. Cerón Viveros
source code

This work integrates image segmentation with adversarial generative training to handle occlusions in façade images. OA-WinSeg uses rectangular boundary guidance to condition the generation of window segmentation masks, capturing structural information about hidden window boundaries. By incorporating adversarial training, rectangular guidance, and bidirectional convolution with 1D kernels, we achieve highly rectangular window shapes, reducing the reliance on complex vectorization mechanisms.
Simulated dataset with synthetic occlusions for window segmentation
Manuela F. Cerón Viveros
source code

This work presents a new method to generate synthetic datasets with occlusions. It enables configuring the percentage of occluded samples, setting size and location constraints for occluding objects. The approach is adaptable across building styles and image resolutions, supports adding new occluding object categories, and facilitates generating datasets of any scale. This dataset addresses challenges in manual occlusion annotation and enables automated real occlusion detection.

Backdoor Attack against NLP models with Robustness-Aware Perturbation defense
Manuela F. Cerón Viveros, Mohammed Maqsood Shaik, GowthamKrishna Addluri

In our research, we address backdoor attacks on deep neural networks (DNNs), where hidden triggers activate malicious behavior while maintaining normal performance on benign data. These attacks exploit vulnerabilities in uncontrolled training scenarios, like third-party datasets or models. Current defenses, such as robustness-aware perturbation-based methods, focus on the robustness gap between clean and poisoned samples. Our approach disrupts this defense by minimizing the robustness gap through adversarial training steps.
Image Synthesis from image-to-image translation of coarsely quantized images
Manuela F. Cerón Viveros, Akshay Dodwadmath

We propose a method for generation of dequantized or finely quantized images. We cast this as an image-to-image translation task using conditional adversarial networks. Using images from a custom dataset with limited data, we demonstrate that our approach can generate finely quantized images, while outperforming the baseline. We further show that our method can be extended to applications such as video regeneration.
